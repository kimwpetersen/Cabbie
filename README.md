# Cabbie

Browser-based arcade taxi game with a 3-act business empire progression. Built as an **AI-first solo production experiment** — Claude Code as sole engineer, all art/audio/localization via AI pipelines, one curator directing.

**Status:** M0 Foundations complete ✅ · M1 (core driving) in progress

**Play the dev build:** https://kimwpetersen.github.io/Cabbie/ *(live after first deploy)*

## DNA

Lane Splitter (driving feel) × Mujaffa-spillet (city personality) × Lula: The Sexy Empire (three-act progression).

## Tech

Three.js r128 · Vite · vanilla JS ES modules · Vitest · Web Audio API · localStorage save
